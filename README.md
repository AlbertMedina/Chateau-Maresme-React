## Installation
1. Clone repository (https://github.com/AlbertMedina/Chateau-Maresme-React.git).
`git clone https://github.com/AlbertMedina/Chateau-Maresme-React.git`
2. Import project to your IDE (optional).

## Execution
1. Navigate into the project folder.
`cd <PROJECT_NAME>`
2. Install dependencies.
`npm install`
3. Start the development server.
`npm run dev`
4. Open it in your browser.
`http://localhost:5173`
