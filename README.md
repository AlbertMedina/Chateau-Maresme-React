The website is **deployed on Netlify** at: https://chateau-maresme-react.netlify.app

## Installation & Setup
1. Clone repository (https://github.com/AlbertMedina/Chateau-Maresme-React.git).
`git clone https://github.com/AlbertMedina/Chateau-Maresme-React.git`
2. Navigate into the project folder.
`cd Chateau-Maresme-React`
3. Install dependencies.
`npm install`
4. Start the development server.
`npm run dev`
5. Open it in your browser.
`http://localhost:5173`
